1 - git status
2 - git add [files] - добавляет файлы в stage
3 - git commit -m 'comment'
4 - git log / git log --oneline - показывают информацию о записях, комитах
5 - git push [rep_link] [branch_name]
6 - git remote -v - показывает ссылку на репазиторий
7 - git branch - показывает на какой ветке на данный момент находитесь
